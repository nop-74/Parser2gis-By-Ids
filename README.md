# Parser2gis-By-Ids
Парсер организаций по ID фирмы из 2GIS 2026
![Alt text](https://github.com/nop-74/Parser2gis-By-Ids/blob/main/full.png)
- На сайте 2GIS.Ru содержится справочник организаций, насчитывающий около 3млн организаций. Это один из самых актуальных справочников организаций в - настоящее время — сотрудники этой компании регулярно обзванивают все организации, выявляют изменения контактных данных и актуализируют базу - -данных. Актуальность данных крайне важна, поскольку неактуальные данные увеличивают ваши затраты.

- Если у Вас есть выгрузка из 2гис, сделанная нашим парсером, или есть одна из наших готовых баз, и Вам необходимо её актуализировать, Вам - необходим этот парсер.

- Иногда сайт 2гис в ответ на запросы выдаёт рекапчу. Обычно такое бывает, когда страна, из которой происходит парсинг, отличается от выгружаемой - страны. Парсер ДубльГИС умеет работать с сервисом автоматического распознавания капчи для разгадки выдаваемых сайтом рекапч. Вам необходимо иметь - аккаунт в этом сервисе для пользования парсером. Минимального баланса в 1$ хватит на несколько месяцев использования антикапчи, в стандартных условиях.

- На вход парсеру требуется задать ID фирм или ссылки на карточки компаний, примеры корректных входных данных: ‘563478234823896; 70000001025624768; https://2gis.ru/firm/563478234623429; https://2gis.ru/barnaul/firm/70000001028579991 — ссылки могут содержать город, но не обязательно.

- В списке может быть несколько тысяч ID или ссылок, парсер обработает их все по очереди.

- В результате работы программы Вы получаете таблицу в Excel с информацией о компании с заданным ID.

# У ВАС ЕСТЬ ДВА ВАРИАНТА
# РАБОТЫ С КОНТАКТАМИ ОРГАНИЗАЦИЙ ИЗ 2ГИС:

- Разработанная нами программа-парсер Parser2gisByIds производит выгрузку всех организаций по заданным ссылкам или ID.

- Программа умеет приостанавливать свою работу, и продолжать её. Паузу можно включать вручную, а если отключится интернет — программа поставит себя - на паузу автоматически. Если во время сбора на сайте возникнет ошибка, и парсер не сможет получить данные — она поставит себя на паузу на - - некоторое время, затем автоматически проснется и сделает еще несколько попыток получить те же данные.

- Индикатор прогресса внизу показывает ход выполнения работы.
